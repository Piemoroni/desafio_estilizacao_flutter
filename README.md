# Flutter Funcionários

Aplicativo desenvolvido em Flutter que exibe uma lista de funcionários a partir de um arquivo JSON local (mockup), com navegação entre registros e interface estilizada.

---

## Objetivo

O projeto tem como objetivo praticar:

- Leitura de arquivos JSON locais
- Manipulação de listas no Flutter
- Navegação entre dados (anterior/próximo)
- Uso de Dropdown
- Organização de projeto (UI, root, assets)
- Criação de Splash Screen com animação
- Customização de tema e cores

---

## Funcionalidades

Splash Screen com animação (fade + scale)  
Carregamento de dados via JSON local  
Exibição de funcionários em formato de card  
Navegação entre funcionários (Anterior / Próximo)  
Botões desabilitados no início e fim da lista  
Dropdown para seleção direta de funcionário  
Imagens locais (assets)  
Layout padronizado e responsivo
